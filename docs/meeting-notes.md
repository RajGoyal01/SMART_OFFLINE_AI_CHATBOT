## 📅 Day 1 – 

### Objectives
- Identify and finalize required hardware components for the prototype device.

### Discussion Points
- Selection between Raspberry Pi 4 and ESP32 as the primary controller.
- Power supply and cabling requirements.
- HDMI cable and display configuration.
- Storage requirement (32GB SD card).
- GPS module integration.
- LoRa communication module selection.

### Decisions Taken
The following components were finalized for procurement:

- Raspberry Pi 4 Model B (1GB RAM)
- LoRa Module RA-02 (433 MHz Wireless Spread Spectrum Transmission)
- 3.5” Touch Screen LCD Display for Raspberry Pi
- NEO-6M GPS Module with EPROM (Normal Quality)
- 433 MHz 2.5 dBi Omnidirectional Folding Antenna
- 20 cm SMA Female to IPEX1 Connector Cable (Thread 13mm RG1.13)

### Action Items
- Place orders for all finalized hardware components.
